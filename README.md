# Curso de Design de Sistemas
Curso em português sobre **System Design**, arquitetura de software e sistemas distribuídos.
O objetivo é entender como os principais componentes funcionam, quando utilizar cada abordagem e quais trade-offs existem em cada decisão arquitetural.

## Como estudar

Siga os módulos na ordem.

Ao finalizar cada módulo, tente responder:

- Qual problema esse componente resolve?
- Quando ele deve ser utilizado?
- Quais alternativas existem?
- Quais são seus principais riscos?
- Como ele escala?
- Como ele se comporta em caso de falha?
